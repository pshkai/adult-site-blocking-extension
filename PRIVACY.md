# Privacy Policy

Never Give Up Site Blocker does not collect, sell, or share personal user data.

The extension downloads a public adult-domain hosts list and stores the parsed blocklist locally in the browser. It checks top-level navigation locally so blocked domains can be redirected to the bundled encouragement page. This processing stays on the user's device.

The extension does not collect personally identifiable information, health information, financial information, authentication information, personal communications, location data, web history, user activity, or website content.

The extension embeds a YouTube video on the local blocked page. YouTube may handle playback according to Google's own privacy policies when the video is loaded.

For support, visit the GitHub repository:
https://github.com/pshkai/adult-site-blocking-extension
